> 困难算法题，最长递增子序列，使用 JavaScript 语言编写

## 题目

给你一个整数数组 nums，找到其中最长严格递增子序列的长度。

子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列。

### 输入格式

1 <= nums.length <= 2000
-10000 <= nums[i] <= 10000

### 输出格式

最长严格递增子序列的长度

### 输入样例

```plaintext
10 9 2 5 3 7 101 18
```

### 输出样例

```plaintext
4
```
