> 困难算法题，超级回文数，使用 JavaScript 语言编写

## 题目

如果一个正整数自身是回文数，而且它也是一个回文数的平方，那么我们称这个数为超级回文数。

现在，给定两个正整数 L 和 R ，请按照从小到大的顺序打印包含在范围 [L, R] 中的所有超级回文数。

注：R包含的数字不超过20位

回文数定义：将该数各个位置的数字反转排列，得到的数和原数一样，例如676，2332，10201。

### 输入格式

L,R。例如4,1000

### 输出格式

[L, R]范围内的超级回文数，例如[4, 9, 121, 484]

### 输入样例

输入样例一：

```plaintext
4,1000
```

### 输出样例

```plaintext
[4, 9, 121, 484]
```
